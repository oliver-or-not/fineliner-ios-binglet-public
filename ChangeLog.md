# Change Log

## marketing version 1.0.3; build number 10; 2026. 3. 2.
- 게임 UI를 개선했다.
    - 다크 모드에서 블록된 노드 위치 표시를 더 밝게 했다.
    - 이펙트 시간을 줄였다.
    - 링크 끝에 corner radius를 추가했다.
    - 노드 블록 표시를 X에서 nosign으로 변경했다.
    - 배치를 담당하는 버튼을 액티브 빙글렛 컨테이너 주변에 추가했다.
    - 놓을 위치가 몇 개 안 남으면 깜빡거리면서 힌트로 보여주도록 했다.
- 튜토리얼을 개선했다.
    - UI 변경사항을 튜토리얼에 반영했다.
    - 튜토리얼 반짝임 강조 표시가 더 빠르게 반짝이도록 했다.
    - 반짝임 강조 표시를 OK 버튼에도 추가했다.
    - 튜토리얼에서 빙글이 뭔지 설명했다.
    - 자유롭게 놓아보라는 내용, 그리고 놓을 자리가 없으면 게임이 끝난다는 내용을 설명에 추가했다.
- 최종 점수 화면에서 뒤로 가기 버튼을 없애고 공유 버튼을 추가했다.

## marketing version 1.0.2; build number 7; 2026. 2. 14.
- 다크 모드에서 게임 보드의 노드 shade를 더 진하게 수정했다.
- 튜토리얼 중 다이얼로그의 버튼을 누를 차례에 게임 보드를 조작할 수 없도록 변경했다.
- 영구 저장소를 사용하는 방법으로서 user defaults 대신 파일 매니저를 쓰도록 변경했다.
    - 마이그레이션 포함.
- 한국어와 일본어의 번역을 개선했다.
    - 빙글과 빙글렛 표현을 영어에서 현지 언어로 변경했다.
- 메인 화면 버튼, 설정 화면 섹션 타이틀 폰트 크기를 수정했다.
    - 22 -> 20
